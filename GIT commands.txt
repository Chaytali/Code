Firstly do git bash here from where you want to push the data
check the status with git status
git init
Add data using git add .
Commit git commit -m "Initial commit",m stands for message
Add the remote repository name
git remote add origin "Github repository name"
git push origin master

To change the master
git remote set-url origin "Github repositoty name"

If you are having a readme.txt file in the repository do git pull origin master
git push -f origin master where f stands for forcefully
